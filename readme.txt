== Description ==

This plugin is created for easy adding files from <a target="_blank" href="http://disk.yandex.com/">Yandex Disk</a> service to posts or pages of your wordpress site.

== Installation ==

=To install the plugin=

1. Upload `wp-yadisk-files` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set Yandex login and password settings on Yadisk Files settings page

=To use the plugin=

1. Go to post editor
1. Click Yadisk Files button
1. Choose file from "Choose File" popup
1. Save post/page
